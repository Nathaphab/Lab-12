# 📱 ชุมชนอุ่นใจ (Oonjai Contacts)

## 🎯 จุดประสงค์แอป
แอปพลิเคชันสำหรับรวบรวมเบอร์โทรศัพท์ฉุกเฉินและหน่วยงานสำคัญในชุมชน เพื่อให้คนในพื้นที่สามารถติดต่อขอความช่วยเหลือได้ทันที

## 📋 รายละเอียดโปรเจกต์
- **Framework:** Flutter
- **State Management:** Provider
- **Local Database:** SQFlite
- **ฟีเจอร์:** เพิ่มรายชื่อ, ลบรายชื่อ, เก็บข้อมูลถาวรในเครื่อง
